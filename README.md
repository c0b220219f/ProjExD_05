# ゲーム のタイトル
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公キャラクターを十字キーなどで行動決定し、戦闘などを行う

## ゲームの実装
###共通基本機能
* プレイヤーの枠
* ボタンの配置
* 敵の初期配置
### 担当追加機能
* 敵の攻撃（担当：川地）
* プレイヤーの動き（担当：大石）
* プレイヤーの選択と行動(戦う、道具)（担当：上田）：コマンド選択でアイテムの使用や攻撃画面に移行
* プレイヤーの選択と行動(話す、見逃す)（担当：上田）：コマンド選択で敵と会話や逃げる画面に移行
* ライフ管理(UIの表示)（担当：小川）
* BGMと敵キャラ（担当：小川）
